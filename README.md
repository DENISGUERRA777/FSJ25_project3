# Guía de Ejercicios JS

A collection of 10 interactive JavaScript exercises built with vanilla HTML and JavaScript. Each exercise demonstrates fundamental programming concepts through a simple browser-based UI.

## Exercises

| # | Description |
|---|-------------|
| 1 | **Age checker** – Determines whether the entered age is under or over 18. |
| 2 | **Grade calculator** – Computes a student's final grade from exam (20%), tasks (40%), attendance (10%) and research (30%) scores. |
| 3 | **Salary raise calculator** – Calculates the salary increase for an employee based on their category (A = 15%, B = 30%, C = 10%, D = 20%). |
| 4 | **Largest number** – Displays the greater of two entered integers. |
| 5 | **Ford car selector** – Shows a 5% discount message for the chosen Ford model (Fiesta, Focus or Escape). |
| 6 | **Travel destination** – Displays the discount percentage for the selected destination (La costa del Sol / La Palma = 5%, Panchimalco = 10%, Puerto el Triunfo = 15%). |
| 7 | **Number analyser** – Reads 10 integers and reports the count of negatives, positives, multiples of 15, and the sum of even numbers. |
| 8 | **Multiplication table** – Generates the multiplication table (1–10) for the selected number. |
| 9 | **Temperature converter** – Converts a Celsius value to Fahrenheit and classifies it as low, adequate, or high. |
| 10 | **Class shift averages** – Calculates the average age of students in the morning, afternoon and evening shifts, and identifies which shift has the highest average. |

## Getting Started

No build tools or dependencies are required. Simply open `index.html` in any modern web browser.

```bash
# Clone the repository
git clone https://github.com/DENISGUERRA777/FSJ25_project3.git

# Open the page
cd FSJ25_project3
open index.html   # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```
